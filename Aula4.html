O TSX nos ajuda em produtividade

o tsx faz o node "entender" o typescript

no final precisaremos TRANSPILAR o código

Resumindo:

Início: npm init -y
Instalar: npm install -D typescript


Agora digite: para instalar o tsx

npm i -D tsx

Agora para RODAR O CÓDIGO usa-se apensas

npx tsx (seu módulo TS)
